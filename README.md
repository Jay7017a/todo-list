# todo-list
Python and MySQL based To-Do List application with CRUD functionality
A simple Python-based To-Do List application implementing CRUD operations.
Allows users to add, view, update, and delete tasks.
Focuses on backend logic and clean code structure.
1. Clone the repository
2. Run the Python script: python todo.py
3. Follow menu options to manage tasks
Features:
- Add task
- View tasks
- Update task
- Delete task
